## QuickStart

### Linux
   1. ```git clone ```
   2. ```cd Muzendis```
   3. execute sql script (create&fill db)
   4. ```dotnet restore```
   5. ```dotnet run```

### Windows
   1. ```git clone ```
   2. ```cd Muzendis```
   3. execute sql script (create&fill db)
   4. open in VS ```Muzendis.sln```
   5. Build&Run
